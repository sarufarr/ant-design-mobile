### 属性

| 参数    | 说明   | 类型     | 默认值 |
| ------- | ------ | -------- | ------ |
| percent | 百分比 | `number` | `0`    |

### CSS 变量

| 属性          | 说明       | 默认值    | 全局变量                         |
| ------------- | ---------- | --------- | -------------------------------- |
| --track-width | 线条宽度   | `3px`     | `--adm-progress-bar-track-width` |
| --fill-color  | 填充的颜色 | `#1677FF` | `--adm-progress-bar-fill-color`  |
